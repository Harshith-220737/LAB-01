# WTLAB_N220737 - Web Technologies Lab Collection

A collection of web development labs and projects from college coursework. Includes learning exercises, cybersecurity awareness projects, authentication systems, and a professional portfolio.

## Folder Structure

### Learning & Lab Exercises

- **class_01-05-26** - Basic JavaScript DOM manipulation with event listeners
- **class_22-12-25** - HTML tables and CSS styling practice
- **LAB-01** - DOM manipulation with media player integration
- **LAB-02_Cyber-AwareUI** - Cybersecurity awareness website
- **LAB-03** - Cybersecurity tips and educational content
- **LAB-04** - User registration system with MySQL
- **LAB-05** - File upload and download system

### Applications

- **customer_support** - Google OAuth 2.0 authentication system
- **customer_support_git** - GitHub OAuth 2.0 authentication system
- **customer_support_MongoDB** - Production OAuth system with MongoDB database

### Portfolio

- **portfolio** - Professional cybersecurity portfolio with responsive design and theme switching

## Quick Start

Each folder is self-contained. Check individual README files in each folder for specific setup instructions.

## Technologies Used

- HTML, CSS, JavaScript
- PHP with MySQL and MongoDB
- OAuth 2.0 (Google, GitHub)
- Bootstrap 5
- Composer (PHP dependencies)
