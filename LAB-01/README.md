# LAB-01 - DOM Manipulation & Media

Comprehensive lab project demonstrating JavaScript DOM manipulation, event handling, and media integration.

## What's Here

- `index.html` - HTML page with multiple interactive elements
- `index.js` - JavaScript functions for DOM manipulation
- `Vennilavu Saaral.mp3` - Sample audio file
- `sahana-rajasab.mp4` - Sample video file

## Features

- DOM element manipulation (text, styles, classes)
- Event listeners and handlers
- Form validation
- Text content updates
- Hover effects
- Audio and video player integration
- Background color changes

## How to Use

1. Open `index.html` in a web browser
2. Interact with buttons and form elements
3. Test audio/video playback
4. Open console (F12) to see any debug information

This lab covers important JavaScript fundamentals for web interactivity.
