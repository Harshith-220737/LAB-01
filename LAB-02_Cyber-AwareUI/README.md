# LAB-02 - Cybersecurity Awareness UI

Educational website promoting cybersecurity awareness with responsive design and styled components.

## What's Here

- `cyber.html` - Complete cybersecurity awareness page
- `styles.css` - Responsive CSS with card layouts

## Features

- Responsive navigation header
- Hero section with main message
- Tips cards for security best practices
- Threats and vulnerabilities section
- Mobile-friendly responsive design
- Clean card-based layout

## Topics Covered

- Strong password creation
- Two-factor authentication (2FA)
- Phishing attack awareness
- Software updates importance
- Safe browsing practices
- Public Wi-Fi risks

## How to Use

1. Open `cyber.html` in a web browser
2. Browse through the information sections
3. View on different screen sizes to see responsive design

Great example of combining HTML and CSS for an educational website.
