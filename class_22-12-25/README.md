# class_22-12-25 - HTML Tables & CSS

Practice project for HTML tables, iframes, and CSS styling.

## What's Here

- `index.html` - Main page with tables and embedded iframe
- `external.css` - External stylesheet for table styling

## Features

- HTML tables with multiple columns
- Embedded iframe showing an external website
- CSS styling with different table layouts
- Personal and faculty information tables

## How to Use

1. Open `index.html` in a web browser
2. View the styled tables
3. See the embedded content in the iframe

Good reference for table styling and layout techniques.
