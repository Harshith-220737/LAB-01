# Professional Portfolio Website

A modern, responsive cybersecurity portfolio website with dark/light theme switching and smooth animations.

## What's Here

- `index.html` - Main portfolio page
- `assets/` - CSS, JavaScript, and images
- `pages/` - Additional pages (about, contact, experiments)
- `data/` - Content and project data

## Features

- Responsive design with Bootstrap 5
- Dark/light theme toggle
- Smooth scroll animations
- Professional layout
- Navigation menu
- Project showcase
- Skills section
- Contact information
- About/Bio section
- Scroll progress indicator
- Mobile-friendly interface

## Folder Structure

```
portfolio/
├── index.html              # Main page
├── assets/
│   ├── css/
│   │   ├── main.css       # Additional styles
│   │   └── portfolio.css  # Main portfolio styles
│   ├── js/
│   │   ├── app.js         # Core functionality
│   │   ├── portfolio.js   # Portfolio interactions
│   │   └── data.js        # Project/skill data
│   └── images/            # Portfolio images
├── pages/
│   ├── about.html         # About page
│   ├── contact.html       # Contact page
│   ├── experiments.html   # Experiments list
│   └── experiment-details.html # Experiment details
└── data/                  # Content files
```

## How to Use

1. Open `index.html` in a web browser
2. Browse through different sections
3. Click on projects to see details
4. Toggle dark/light theme using the theme button
5. Use navigation menu to jump to sections

## Technologies Used

- HTML5
- CSS3 with responsive design
- JavaScript (vanilla)
- Bootstrap 5
- Font Awesome icons
- Google Fonts

## Theme Switching

The portfolio supports both dark and light themes. Your theme preference is saved in browser localStorage.

## Customization

- Edit project data in `assets/js/data.js`
- Modify colors in `assets/css/portfolio.css`
- Add new pages in the `pages/` directory
- Replace images in `assets/images/`

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
