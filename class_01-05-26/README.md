# class_01-05-26 - JavaScript Basics

Simple JavaScript and HTML project for learning DOM manipulation and event handling.

## What's Here

- `myhtml.html` - HTML page with buttons and paragraphs
- `myscript.js` - JavaScript code for button clicks and DOM updates

## Features

- Button click event listeners
- Text content changes
- Background color manipulation
- Basic HTML structure

## How to Use

1. Open `myhtml.html` in a web browser
2. Click buttons to see JavaScript in action
3. Check the console (F12) for any messages

This is a beginner-level exercise for learning how JavaScript interacts with HTML elements.
