# LAB-03 - Cyber-World Tips

Educational website with cybersecurity tips and best practices.

## What's Here

- `main.html` - Cybersecurity tips and educational content
- `edit.css` - Styling for the page layout

## Features

- Navigation navbar
- Multiple security tips sections
- Tips on secure passwords
- Public Wi-Fi safety
- Two-factor authentication benefits
- Software update importance
- Responsive design

## How to Use

1. Open `main.html` in a web browser
2. Read through the cybersecurity tips
3. Share with others to promote security awareness

This is an educational resource designed to teach basic cybersecurity practices.
